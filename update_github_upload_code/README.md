# 👟 Dhankor Shoe Store

A modern **premium shoe e-commerce website** built using **HTML, CSS, and JavaScript** with advanced UI animations and interactive features.

---

## 🚀 Live Features

✨ Premium dark glass UI
✨ 3D Hero Section with mouse movement
✨ Animated product cards (Zomato-style)
✨ Add to cart system (with size selection)
✨ LocalStorage (cart + likes saved)
✨ Fully responsive design
✨ Smooth scroll & reveal animations
✨ WhatsApp order integration

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Glassmorphism + Animations)
* Vanilla JavaScript
* LocalStorage API

---

## 🧠 Key Functionalities

### 🛒 Cart System

* Add products with size selection
* Remove items from cart
* Dynamic total calculation

### ❤️ Like System

* Save favorite products
* Stored in browser (LocalStorage)

### 📱 Responsive Design

* Mobile navigation menu
* Adaptive grid layout

### 📦 Order System

* One-click **WhatsApp order integration**
